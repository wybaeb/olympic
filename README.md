## Посчитай кубики

Образовательная игра для развития пространственного мышления и навыков счета у детей.

![Превью игры](preview.png)

### Описание

"Посчитай кубики" - это интерактивная браузерная игра, в которой ребенку предлагается посчитать количество кубиков, расположенных на платформе в изометрической проекции. Игра развивает:

- Пространственное мышление
- Навыки счета
- Внимательность
- Логическое мышление

### Особенности

- Прогрессивная система уровней с увеличивающейся сложностью
- Изометрический 3D-вид для развития пространственного мышления
- Специальная кнопка "присядка" для временного уменьшения высоты кубиков, чтобы увидеть скрытые кубики
- Сохранение прогресса через URL-хеш (можно поделиться ссылкой на конкретный уровень)
- Разноцветные кубики для лучшего визуального восприятия
- Адаптация уровня сложности в зависимости от успехов ребенка

### Как играть

1. Внимательно рассмотрите 3D-сцену с кубиками
2. Посчитайте, сколько всего кубиков на платформе
3. Введите ваш ответ в поле и нажмите "Далее" или Enter
4. Используйте кнопку ⇅, если не уверены, видите ли вы все кубики (временно сжимает высоту для лучшего обзора)
5. После 3-х правильных ответов игра автоматически переходит на следующий уровень

## Решаем уравнения

Образовательная игра для развития логического мышления и навыков решения уравнений.

![Превью игры](equations.png)

### Описание

"Решаем уравнения" - это интерактивная браузерная игра, в которой ребенку предлагается решить систему уравнений с переменными, представленными в виде эмодзи. Игра развивает:

- Логическое мышление
- Навыки решения уравнений
- Внимательность
- Аналитические способности

### Особенности

- Прогрессивная система уровней с увеличивающейся сложностью
- Переменные представлены в виде эмодзи для лучшего восприятия
- Система подсказок для помощи в решении
- Сохранение прогресса через URL-хеш (можно поделиться ссылкой на конкретный уровень)
- Адаптация уровня сложности в зависимости от успехов ребенка

### Как играть

1. Внимательно рассмотрите систему уравнений
2. Определите значения переменных (эмодзи)
3. Введите ваши ответы в соответствующие поля
4. Нажмите "Проверить" или Enter
5. После 3-х правильных ответов игра автоматически переходит на следующий уровень

## Магический квадрат

Образовательная игра для развития математических навыков и логики.

![Превью игры](magic-square.png)

### Описание

"Магический квадрат" — это интерактивная игра, в которой ребенку предлагается заполнить пустые ячейки так, чтобы суммы по всем строкам и столбцам были равны. Игра развивает:

- Математическое мышление
- Навыки сложения и вычитания
- Внимательность
- Логическое мышление

### Особенности

- Прогрессивная система уровней с увеличением размера квадрата и сложности
- Проверка правильности заполнения в реальном времени
- Красочное оформление и удобный интерфейс
- Сохранение прогресса через URL-хеш (можно поделиться ссылкой на конкретный уровень)
- 3 правильных решения — переход на следующий уровень

### Как играть

1. Заполните пустые ячейки числами так, чтобы суммы по всем строкам и столбцам были одинаковыми
2. После заполнения проверьте результат
3. После 3-х правильных решений игра автоматически переходит на следующий уровень

## Узнай время по часам

Образовательная игра для тренировки навыков чтения времени по аналоговым часам.

![Превью игры](clock.png)

### Описание

"Узнай время по часам" — это интерактивная игра, в которой ребенку показывается циферблат с часовой и минутной стрелкой. Необходимо определить и ввести правильные значения часов и минут. Игра развивает:

- Навыки чтения времени по аналоговым часам
- Внимательность
- Концентрацию

### Особенности

- Прогрессивная система уровней с уменьшением шага минутной стрелки (от 15 минут до 1 минуты)
- Крупный циферблат с засечками для каждой минуты
- Автофокус на поле ввода при каждом новом задании
- Сохранение прогресса через URL-хеш (можно поделиться ссылкой на конкретный уровень)
- 3 правильных ответа — переход на следующий уровень

### Как играть

1. Посмотрите на циферблат и определите, сколько сейчас часов и минут
2. Введите значения в соответствующие поля
3. Нажмите "Далее" или Enter
4. После 3-х правильных ответов игра автоматически переходит на следующий уровень

## Соединение монет

Образовательная игра для развития финансовой грамотности и математических навыков.

![Превью игры](coins_preview.png)

### Описание

"Соединение монет" — это интерактивная игра, в которой ребенку предлагается соединять монеты разного номинала для получения нужной суммы. Игра развивает:

- Финансовую грамотность
- Математические навыки сложения
- Распознавание номиналов монет
- Логическое мышление

### Особенности

- Прогрессивная система уровней с увеличивающейся сложностью
- Визуальное представление монет различного номинала
- Интерактивное соединение монет для составления нужной суммы
- Понятная легенда с обозначением всех номиналов
- Адаптация уровня сложности в зависимости от успехов ребенка

### Как играть

1. Внимательно рассмотрите кучки монет на экране
2. Соединяйте монеты, перетаскивая линии от одной кучки к другой
3. Монеты должны быть соединены так, чтобы в каждой паре сумма номиналов равнялась заданной сумме
4. Нажмите "Проверить ответы" после завершения всех соединений
5. После успешного выполнения уровня игра автоматически переходит к следующему заданию

## Технические детали

- Чистый JavaScript без внешних зависимостей
- HTML5 Canvas для отрисовки 3D-сцены и циферблата
- Адаптивный дизайн для различных размеров экрана
- Система сохранения прогресса через URL-хеш

## Установка и запуск

1. Клонируйте репозиторий:
```
git clone https://github.com/username/educational-games.git
```

2. Откройте файл `index.html` в любом современном браузере для доступа к меню игр

Поскольку игры не используют внешние зависимости и серверную часть, они полностью функциональны как есть, без дополнительной настройки.

### Запуск на iPad

Игры могут быть запущены на iPad с помощью приложения [HTML Viewer Q](https://apps.apple.com/ru/app/html-viewer-q/id810042973) или аналогичных приложений для просмотра HTML-файлов:

1. Загрузите файлы в приложение через iTunes File Sharing или iCloud Drive
2. Откройте файл `index.html` в приложении и выберите игру из меню

## Для разработчиков

Игры специально написаны на чистом JavaScript без использования фреймворков, чтобы их можно было легко модифицировать и расширять. Основные компоненты:

- Функции изометрического преобразования координат для игры "Посчитай кубики"
- Алгоритм генерации башен с корректным расположением
- Система проверки и оценки ответов
- Механизм сохранения прогресса через URL-хеш
- Система генерации уравнений для игры "Решаем уравнения"
- Генерация и отрисовка магических квадратов
- Генерация и отрисовка аналоговых часов для игры "Узнай время по часам"
